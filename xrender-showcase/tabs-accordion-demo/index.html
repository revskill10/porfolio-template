Basic Accordion (Single)

This is a **custom widget system** that allows you to create rich, interactive content using simple XML-like syntax. It supports various components including tabs, accordions, cards, and more.
### Using Tabs ```html <div data-widget="tabs"> <tab value="tab1" label="Tab 1">Content</tab> </div> ``` ### Using Accordions ```html <div data-widget="accordion"> <item value="item1" title="Title">Content</item> </div> ```
**Yes!** You can nest any widget inside tabs and accordion items. This includes: - Cards and buttons - Markdown content - Images and code blocks - Even other tabs and accordions

Usage Examples

Learn how to integrate X-Render widgets into your portfolio.

Create flowcharts using JSON schema data attribute:

<div data-widget="xflow" data-height="400px" data-readonly="false" data-schema='{"nodes":[{"id":"start","type":"start","position":{"x":100,"y":100},"data":{"label":"Start"}}],"edges":[]}'> </div>

Create flowcharts using native XML syntax:

<div data-widget="xflow" data-height="400px" data-readonly="false"> <nodes> <start id="start" x="100" y="100" input="Start Process" /> <process id="task" x="300" y="100" input="Process Data" /> <end id="end" x="500" y="100" input="End Process" /> </nodes> <edges> <edge source="start" target="task" /> <edge source="task" target="end" /> </edges> </div>

Generate forms using JSON schema data attribute:

<div data-widget="xform" data-display-type="row" data-label-width="120px" data-schema='{"type":"object","properties":{"name":{"title":"Name","type":"string","required":true}}}'> </div>

Generate forms using native XML syntax:

<div data-widget="xform" data-display-type="column" data-label-width="120px"> <input name="name" type="string" title="Full Name" required /> <input name="email" type="string" format="email" title="Email" /> <select name="role" title="Role"> <option value="user">User</option> <option value="admin">Admin</option> </select> </div>

Create data tables using JSON schema data attribute:

<div data-widget="xtable" data-search="true" data-pagination="true" data-toolbar="true" data-height="400" data-schema='{"columns":[{"title":"Name","dataIndex":"name"}]}'> </div>

Create data tables using native XML syntax:

<div data-widget="xtable" data-search="false" data-pagination="false"> <columns> <text name="name" title="Name" width="150" /> <text name="email" title="Email" width="200" /> <tag name="status" title="Status"> <option value="active">Active</option> <option value="inactive">Inactive</option> </tag> </columns> <data> <row name="John Doe" email="john@example.com" status="active" /> </data> </div>

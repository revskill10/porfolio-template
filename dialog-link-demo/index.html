Technical Features

## Dialog Link Widget Features ### Content Types - **`portfolio`**: Loads `index.html` + `index.json` from GitHub (default) - **`markdown`**: Loads and renders markdown files with full formatting - **`html`**: Loads raw HTML content ### Dialog Sizes - **`sm`**: Small dialog (max-w-md) - **`md`**: Medium dialog (max-w-lg) - **`lg`**: Large dialog (max-w-2xl) - **`xl`**: Extra large dialog (max-w-4xl) - **`full`**: Full screen dialog (max-w-7xl) ### Trigger Options - **Button triggers**: Use `trigger-variant` and `trigger-size` - **Custom triggers**: Wrap any content as the trigger - **Styling**: Full CSS class support ### Path Resolution - **Relative paths**: `./about`, `../blog/post.md` - **GitHub integration**: Loads from raw.githubusercontent.com - **Error handling**: Graceful fallbacks with retry options ### Performance - **Lazy loading**: Content loads only when dialog opens - **Caching**: Content cached after first load - **Async**: Non-blocking UI with loading indicators
